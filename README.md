# EzBt (Easy Bluetooth)

EzBt is a modern Home Assistant integration and frontend panel for discovering and pairing Bluetooth devices (speakers and HID devices) by their friendly names.

## Features
- **Modern UI**: Apple-esque design with rounded corners and soft shadows.
- **Discovery**: Real-time scanning for Bluetooth peripherals.
- **Easy Pairing**: Simple one-click connection for speakers and HID devices.
- **HACS Compatible**: Easily installable via the Home Assistant Community Store.

## Installation
1.  Ensure you have a functional Bluetooth adapter configured in Home Assistant.
2.  Install this integration via HACS as a custom repository.
3.  Add "EzBt" from the Integrations menu.
4.  Access the "EzBt" panel from your sidebar.

## Usage
Click "Scan for Devices" to discover nearby Bluetooth peripherals. Use the "Connect" button on each card to pair with your preferred device.
